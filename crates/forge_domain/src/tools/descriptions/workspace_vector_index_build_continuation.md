Agent-invoked vector index build continuation for semantic search readiness repair. Explicit only: this tool is never called by `sem_search`, diagnostics, automatic context injection, or tool-description rendering. Use only when a prior diagnostic indicates that the current workspace has a fresh manifest and a vector build is the safe next action.

Execution is typed and service-owned: canonicalizes and validates the workspace root, runs a preflight diagnostic, builds only for `vector_build_suggested` with `safe_to_suggest_build=true`, calls the workspace vector-index build service directly, runs a post-build diagnostic, and returns a closed typed status. It never executes diagnostic display command text, never shells out, and never deletes, truncates, or cleans corrupt/ambiguous vector artifacts.
